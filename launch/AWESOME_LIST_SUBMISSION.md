## Longer listing

Howdex is an open verification layer for agent know-how. It turns agent
execution traces into portable, receipt-backed procedures that can be rendered
as operational guidance. It is local-first by default, stores memory in SQLite,
supports a local Codex JSON format, and includes an MCP server plus optional
adapters for common agent runtimes.
