# Howdex pilot tracking

This file tracks real external outreach after Ross manually posts links or
receives external contributions. It starts at zero by design.

Do not count internal dogfood, private intent, likes, upvotes, or vague interest
as confirmed pilot usage.

## External adoption counter

- external_pilot_users_confirmed: 0
- external_procedure_submissions: 0
- external_verified_receipts: 0
- external_repos_using_howdex: 0

## Evidence rules

- A pilot user counts only after a real external person or team confirms use.
- An external issue or PR counts only with a public link or recorded private
  evidence Ross chooses to track.
- An external procedure submission counts only when someone outside the project
  submits a procedure entry, trace, or comparable operational memory artifact.
- An external verified receipt counts only when the receipt is inspectable and
  tied to an external procedure or external Howdex use.
- An external repo counts only when a repo outside this project visibly uses
  Howdex or documents an integration.
